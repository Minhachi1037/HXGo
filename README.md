### 0x02 Build HXGo Windows Client

First of all, you need to install NodeJS platform from [NodeJS](https://nodejs.org/en/download) with NPM

After install, cd to HXGo Folders, enter to install dependence:

> npm install express -y

Wait for success, type the command down to start debug server:

> node hxgo.js

If it return port IP at 10374, that means succeeded.

View **localhost:10374** to use test-server

**Release Version Donot Need to build!**
